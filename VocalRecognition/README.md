# Vocal Recognition
Android app : Vocal Recognition , with WebView.
A proof of concept for VocalRecognition SpeechAPI with android and Webview.

## Download
```sh
$ git clone https://github.com/ctrlfagency/VocalRecognition.git ctrlfagency

```
